# Android Mod Menu Auto Injector

Automation tool for injecting Android mod menu into Unity IL2CPP games.

Features:
- Automatic Manifest Modification
- Smali Injection
- Mod Menu Service Setup
- Permission Injection
- Library Copying

Usage:

1. Decompile APK
2. Open tool
3. Select APK folder
4. Click Inject

The tool will automatically modify:

- AndroidManifest.xml
- Main Activity smali
- Required mod menu files
